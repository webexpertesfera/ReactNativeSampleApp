## Installation & Setup

Clone the Repository
Start by cloning the repository to your local machine:

## git clone https://github.com/webexpertesfera/ReactNativeSampleApp.git
cd your-project-name
Install Dependencies
Once inside the project directory, you need to install the project dependencies.


Using npm:
```bash
npm install
```
Using Yarn:

```bash
yarn install
```
Running the Project

Option 1: Run on Android Emulator or Device
Open Android Studio and ensure that the Android Emulator is running or connect an Android device via USB.
Run the following command:
```bash
npx react-native run-android
```
This will build the app and launch it on the Android emulator or physical device.
Option 2: Run on iOS Simulator (macOS only)
Ensure you have Xcode installed on your macOS machine.
Start the iOS simulator:
npx react-native run-ios
This will build the app and launch it on the iOS simulator.
